## 项目经历

### DiskDoctor — 磁盘体检医生（2026.06）
*独立开发 | Python + PySide6 + FFmpeg*

- 开发了一款 Windows 磁盘空间分析工具，支持全盘扫描、文件分类统计、重复文件检测
- 实现了**感知哈希（pHash/dHash）近重复图片检测**，可识别缩放/裁剪/滤镜后的相似图片，填补了 WizTree/WinDirStat 等同类工具的功能空白
- 集成 DeepSeek/OpenAI/Ollama **三模式 AI 建议引擎**，支持在线 API 和本地大模型
- 设计**自定义表情包系统**和 Treemap/Donut/Bar 三种可视化视图
- 支持中英文双语切换、备份恢复、快照对比、系统文件多重删除保护
- 技术栈：Python 3.14 + PySide6 + SQLite + imagehash + FFmpeg

### RenderSafe — 视频渲染断点续传（2026.06）
*独立开发 | Python + FastAPI + FFmpeg | [开源](https://github.com/zjw0420/-rendersafe)*

- 设计了分段渲染+检查点机制，解决了长视频导出时因崩溃导致全部重来的痛点
- 每完成一段自动存档，中断后自动从断点继续，无需手动干预
- 前端使用纯 HTML/CSS/JS 实现进度可视化，后端使用 FastAPI + 后台线程处理渲染
- 已发布到 GitHub 并采用 MIT 协议开源

### ClipMate — AI 视频剪辑决策助手（2026.06）
*独立开发 | Python + FastAPI + DeepSeek API*

- 改变了传统 AI 剪辑"黑盒输出"的范式，让 AI 生成可审查、可修改的编辑决策列表
- 用户对每条 AI 建议进行审批/拒绝/修改，而非被动接受自动生成的结果
- 后端采用 FastAPI + SQLite，前端纯原生 JS 实现 SPA 交互
- 支持 9 种剪辑决策类型（切入/切出/转场/B-roll/文字叠加/音乐节拍等）

### SnapCut — 智能视频处理工具箱（2026.06）
*独立开发 | Python + FastAPI + FFmpeg*

- 实现了视频分析自动建议系统，上传视频后自动检测无声段落、文件大小、横竖屏比例
- 提供可视化时间轴（缩略图+静音标记+场景切换标记），支持 I/O 键设入出点裁剪
- 集成 6 种常用视频处理功能：去无声段、压缩、裁剪、提取音频、格式转换、横竖屏适配
